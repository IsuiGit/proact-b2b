# WARMUP — рубрика писем прогрева

## Нормализация названий
Сбербанк = ПАО Сбербанк = Сбер = Sberbank = Sber — одна компания. ПАО Сбербанк и алиасы исключены (не являются клиентами).

## Порог включения
- `confidence ≥ 70%`
- Максимум 5 писем
- Сортировка по confidence убывание

## Структура письма (строго)

### 1. Тема
Формат: `[Событие] на [сумма] — [продукты] от Сбербанка`
Пример: «Обеспечение закупки на 150 млн ₽ — гарантии и финансирование от Сбербанка»

### 2. Обращение
```
Уважаемый руководитель [Название компании],
```

### 3. Хук (1 абзац, 1–2 предложения)
Упоминание конкретного события + сумма. Без общих фраз.
Пример: «Мы видим, что ваша компания провела закупку на 150 миллионов рублей.»

### 4. Ценностное предложение — основной продукт (2–3 предложения)
Конкретный продукт Сбербанка + конкретные условия/сроки.

Маппинг продуктов → формулировки:
- **Банковские гарантии**: «Оформление — от одного рабочего дня, ставки — одни из лучших на рынке»
- **Кредитование**: «Лимит может быть согласован заранее, чтобы вы могли пользоваться им по мере необходимости»
- **РКО**: «Приоритетное обслуживание, расширенные лимиты, интеграция с учётными системами через API»
- **ВЭД**: «Валютный контроль, аккредитивы, хеджирование валютных рисков — всё в одном окне»
- **Факторинг**: «Финансирование дебиторской задолженности без ожидания оплат от контрагентов»
- **Лизинг**: «Лизинг оборудования/транспорта с сезонным графиком платежей»

### 5. Дополнительный продукт (1–2 предложения)
Второй продукт из рекомендованных в SCOUT. Кратко, с конкретным преимуществом.

### 6. CTA (1 предложение)
Предложение встречи/звонка на 20–30 минут. С обещанием подготовки расчётов.
Пример: «Предлагаю провести короткую встречу — 20–30 минут — и определить, какие инструменты подходят. Я подготовлю предварительный расчёт до встречи.»

### 7. Подпись
```
С уважением,
Персональный менеджер, Сбербанк
```

## Правила
- Объём: 150–250 слов (считать)
- Язык: русский
- Тон: профессиональный, без канцелярита
- **Без код-блоков, JSON, технических артефактов**
- Каждое письмо начинается с `### Письмо для [Компания]`
- Между письмами — разделитель `---`
